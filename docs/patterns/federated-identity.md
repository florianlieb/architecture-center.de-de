## <a name="solution"></a><span data-ttu-id="d25c0-119">Lösung</span><span class="sxs-lookup"><span data-stu-id="d25c0-119">Solution</span></span>

<span data-ttu-id="d25c0-120">Implementieren Sie einen Authentifizierungsmechanismus, der Verbundidentität verwenden kann.</span><span class="sxs-lookup"><span data-stu-id="d25c0-120">Implement an authentication mechanism that can use federated identity.</span></span> <span data-ttu-id="d25c0-121">Trennen Sie die Benutzerauthentifizierung vom Anwendungscode, und delegieren Sie die Authentifizierung an einen vertrauenswürdigen Identitätsanbieter.</span><span class="sxs-lookup"><span data-stu-id="d25c0-121">Separate user authentication from the application code, and delegate authentication to a trusted identity provider.</span></span> <span data-ttu-id="d25c0-122">Dies kann die Entwicklung vereinfachen und Benutzern die Authentifizierung über eine größere Anzahl an Identitätsanbietern (IdP) ermöglichen, während gleichzeitig der Verwaltungsaufwand minimiert wird.</span><span class="sxs-lookup"><span data-stu-id="d25c0-122">This can simplify development and allow users to authenticate using a wider range of identity providers (IdP) while minimizing the administrative overhead.</span></span> <span data-ttu-id="d25c0-123">Sie können außerdem auf diese Weise die Authentifizierung eindeutig von der Autorisierung entkoppeln.</span><span class="sxs-lookup"><span data-stu-id="d25c0-123">It also allows you to clearly decouple authentication from authorization.</span></span>

<span data-ttu-id="d25c0-125">Die Abbildung zeigt das Verbundidentitätsmuster, wenn eine Clientanwendung auf einen Dienst zugreifen muss, für den eine Authentifizierung erforderlich ist.</span><span class="sxs-lookup"><span data-stu-id="d25c0-125">The figure illustrates the Federated Identity pattern when a client application needs to access a service that requires authentication.</span></span> <span data-ttu-id="d25c0-126">Die Authentifizierung erfolgt durch einen IdP, der zusammen mit einem STS funktioniert.</span><span class="sxs-lookup"><span data-stu-id="d25c0-126">The authentication is performed by an IdP that works in concert with an STS.</span></span> <span data-ttu-id="d25c0-127">Der IdP stellt Sicherheitstoken aus, die Informationen zu dem authentifizierten Benutzer bereitstellen.</span><span class="sxs-lookup"><span data-stu-id="d25c0-127">The IdP issues security tokens that provide information about the authenticated user.</span></span> <span data-ttu-id="d25c0-128">Diese als „Ansprüche“ bezeichneten Informationen schließen die Identität des Benutzers und möglicherweise auch andere Informationen wie z.B. <span data-ttu-id="d25c0-140">Verbundauthentifizierung bietet eine standardbasierte Windows-Lösung für das Problem, Identitäten über unterschiedliche Domänen hinweg zu vertrauen, und kann einmaliges ## <a name="issues-and-considerations"></a><span data-ttu-id="d25c0-149">Probleme und Überlegungen</span><span class="sxs-lookup"><span data-stu-id="d25c0-149">Issues and considerations</span></span>

<span data-ttu-id="d25c0-150">Beachten Sie Folgendes, wenn Sie Anwendungen entwerfen, die Verbundauthentifizierung implementieren:</span><span class="sxs-lookup"><span data-stu-id="d25c0-150">Consider the following when designing applications that implement federated authentication:</span></span>

- <span data-ttu-id="d25c0-151">Authentifizierung kann ein Single Point of Failure sein.</span><span class="sxs-lookup"><span data-stu-id="d25c0-151">Authentication can be a single point of failure.</span></span> <span data-ttu-id="d25c0-152">Wenn Sie Ihre Anwendung in mehreren Rechenzentren bereitstellen, ziehen Sie in Betracht, Ihren Identitätsverwaltungsmechanismus in denselben Rechenzentren bereitzustellen, um Zuverlässigkeit und Verfügbarkeit für Anwendungen zu gewährleisten.</span><span class="sxs-lookup"><span data-stu-id="d25c0-152">If you deploy your application to multiple datacenters, consider deploying your identity management mechanism to the same datacenters to maintain application reliability and availability.</span></span>

- <span data-ttu-id="d25c0-153">Authentifizierungstools ermöglichen das Konfigurieren der Zugriffssteuerung basierend auf den im Authentifizierungstoken enthaltenen Rollenansprüchen.</span><span class="sxs-lookup"><span data-stu-id="d25c0-153">Authentication tools make it possible to configure access control based on role claims contained in the authentication token.</span></span> <span data-ttu-id="d25c0-154">Dies wird häufig als rollenbasierte Zugriffssteuerung (RBAC) bezeichnet und ermöglicht eine feiner abgestufte Steuerung des Zugriffs auf Features und Ressourcen.</span><span class="sxs-lookup"><span data-stu-id="d25c0-154">This is often referred to as role-based access control (RBAC), and it can allow a more granular level of control over access to features and resources.</span></span>
