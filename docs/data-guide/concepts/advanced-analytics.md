## <a name="machine-learning"></a><span data-ttu-id="61088-132">Machine Learning</span><span class="sxs-lookup"><span data-stu-id="61088-132">Machine learning</span></span>

<span data-ttu-id="61088-135">Die Machine Learning-Modellierung wird in der Regel von Datenspezialisten durchgeführt. <span data-ttu-id="61088-139">Machine Learning lässt sich in zwei Hauptkategorien unterteilen:</span><span class="sxs-lookup"><span data-stu-id="61088-139">There are two broad categories of machine learning:</span></span>
